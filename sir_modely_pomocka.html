SIR Modely

ZÁKON ZACHOVANIA ĽUDÍ:
Súčet všetkých plusov a mínusov v jednom modeli musí byť vždy NULA!
[ N(t+1) = S(t) + I(t) + R(t) ... = konštanta ]

[1] Štyri stavebné kocky (Toky)

1. Nákaza (Infekcia)
β * S * I / N
- S + I
2. Uzdravenie (Recovery)
γ * I
- I + R (alebo do S ak SIS)
3. Smrť (Death)
γD * I
- I + D
4. Očkovanie (Vaccine)
v * S
- S + V

[2] Modely (LEGO skladačka)

Model SIR (Klasika)
S → I → R
S = S - (β*S*I/N)
I = I + (β*S*I/N) - (γ*I)
R = R + (γ*I)
Model SIS (Bez imunity)
S ⇄ I
S = S - (β*S*I/N) + (γ*I)
I = I + (β*S*I/N) - (γ*I)
Model SIRD (S úmrtiami)
S → I → R | I → D
S = S - (β*S*I/N)
I = I + (β*S*I/N) - (γR*I) - (γD*I)
R = R + (γR*I)
D = D + (γD*I)
Model SISV (S očkovaním)
V ← S ⇄ I
S = S - (β*S*I/N) + (γ*I) - (v*S)
I = I + (β*S*I/N) - (γ*I)
V = V + (v*S)

[3] Výpočty Parametrov

  • Miera uzdravenia: Gama (γ) = 1 / dĺžka_choroby_v_dňoch
  • Miera nákazy: Beta (β) = R0 * γ
  • Miera očkovania (v) = Tempo * Účinnosť * (1 - Tí_čo_nechcú)
    (Napr: 0.015 * 0.90 * 0.70)
  • Rozdelenie Gamy pre SIRD:
    γD (Smrť) = γ * úmrtnosť_v_percentách (napr. * 0.20)
    γR (Uzdravenie) = γ * prežitie_v_percentách (napr. * 0.80)
  • Prah kolektívnej imunity (Herd Immunity) = 1 - (1 / R0)
    (Výsledok vynásob 100 pre percentá)
