MATLAB Ťahák: Dátové Typy, Vektory a Matice

0. Základné Príkazy

Základné príkazy pre správu prostredia, zobrazovanie výstupu a prácu so skriptami.

Správa Prostredia

clear
Vymaže všetky premenné z pracovného priestoru (Workspace). Zabraňuje tomu, aby staré hodnoty ovplyvnili nový beh skriptu.
clc
Vymaže text z príkazového okna (Command Window). Výstup je potom čistejší a prehľadnejší.
close all
Zatvorí všetky otvorené okná s grafmi (Figures).
clear;     % Vymaže premenné
clc;       % Vyčistí okno
close all; % Zatvorí grafy

Zobrazovanie Výstupu a Komentáre

disp(hodnota)
Zobrazí (display) hodnotu premennej alebo textu bez vypísania názvu premennej.
; (bodkočiarka na konci riadku)
Potlačí automatický výpis výsledku príkazu do príkazového okna. Kľúčové pre prehľadné skripty.
% (znak percenta)
Označuje komentár. MATLAB ignoruje všetok text od znaku % až po koniec riadku.
vek = 25; % Výstup je potlačený bodkočiarkou
disp('Vek osoby je:');
disp(vek);

% Kombinovaný výpis
disp(['Vek: ', num2str(vek)]);

Pomoc a Dokumentácia

help nazov_funkcie
Zobrazí rýchlu nápovedu k funkcii priamo v príkazovom okne.
doc nazov_funkcie
Otvorí podrobnú oficiálnu dokumentáciu k funkcii v samostatnom okne.
help plot  % Rýchla pomoc pre funkciu plot
doc table % Otvorí plnú dokumentáciu pre table

1. Dátové Typy

1.1 Matica (Matrix)

Základný dátový typ. Všetky prvky musia byť rovnakého typu (napr. double, char). Používajú sa hranaté zátvorky [].

% Definícia matice 2x3
matica = [1, 2, 3; 4, 5, 6];

1.2 Cell Array (Bunkové Pole)

Kontajner pre rôzne typy dát. Každá bunka (cell) môže obsahovať iný dátový typ (číslo, text, maticu...). Používajú sa zložené zátvorky {}.

% Definícia cell array 2x2
data = {'Názov Filmu', 1994; 'Hodnotenie', 9.3};

1.3 Table (Tabuľka)

Štruktúrované dáta podobné databázovej tabuľke. Stĺpce majú názvy. Každý stĺpec má rovnaký typ dát, ale rôzne stĺpce môžu mať rôzne typy.

% Definícia tabuľky z vektorov
mena = {'Jano'; 'Eva'};
vek = [25; 30];
tabulka = table(mena, vek, 'VariableNames', {'Meno', 'Vek'});

1.4 Struct (Štruktúra)

Organizácia dát do pomenovaných polí (fields). Každé pole môže obsahovať ľubovoľný typ dát. Prístup je cez bodkovú notáciu.

% Definícia štruktúry
pacient.meno = 'Peter';
pacient.vek = 51;
pacient.merania = [120, 80];

2. Tvorba a Indexovanie

2.1 Tvorba Matíc a Vektorov

Priame zadanie: Používajú sa hranaté zátvorky [].

  • Prvky v riadku sa oddeľujú medzerou alebo čiarkou.
  • Riadky sa oddeľujú bodkočiarkou.
% Riadkový vektor (pole)
riadok = [1, 2, 3]; % alebo [1 2 3]

% Stĺpcový vektor
stlpec = [1; 2; 3];

% Matica 2x3
matica = [1, 2, 3; 4, 5, 6];

Pomocou funkcií:

A = zeros(2, 3);  % Matica 2x3 plná núl
B = ones(3, 2);   % Matica 3x2 plná jednotiek
C = rand(4, 4);   % Matica 4x4 s náhodnými číslami (0 až 1)
D = eye(3);     % Jednotková matica 3x3 (jednotky na diagonále)

2.2 Generovanie Postupností (operátor :)

Syntax: start:krok:koniec. Ak krok chýba, je automaticky 1.

v1 = 1:5;         % Vráti [1, 2, 3, 4, 5]
v2 = 0:2:8;       % Vráti [0, 2, 4, 6, 8]
v3 = 5:-1:1;       % Vráti [5, 4, 3, 2, 1]

2.3 Prístup k Prvkom (Indexovanie)

MATLAB indexuje od 1. Kľúčové slovo end znamená posledný index.

Matice A(riadok, stlpec)
Používajú sa okrúhle zátvorky ().
matica = [1,2,3; 4,5,6];
prvok = matica(2, 3);         % Vráti 6 (druhý riadok, tretí stĺpec)
cely_riadok = matica(1, :);   % Vráti [1, 2, 3] (prvý riadok, všetky stĺpce)
cely_stlpec = matica(:, 2);   % Vráti [2; 5] (všetky riadky, druhý stĺpec)
posledny = matica(end, end);  % Vráti 6
Cell Array C{riadok, stlpec}
{} (zložené zátvorky): Prístup k obsahu bunky.
() (okrúhle zátvorky): Prístup k bunke ako celku (výsledok je cell array). 
C = {'text', 123; rand(2), [1 2]};
obsah = C{1, 1};  % Vráti 'text' (typ char)
bunka = C(1, 1);  % Vráti {'text'} (typ cell)
Table T.NazovStlpca alebo T{riadky, 'Nazov'}
Bodková notácia .Nazov: Prístup k celému stĺpcu.
{} (zložené zátvorky): Prístup k dátam vo vnútri buniek. 
vek_stlpec = tabulka.Vek;         % Celý stĺpec 'Vek'
vek_jana = tabulka{1, 'Vek'};      % Konkrétna hodnota
prvy_riadok = tabulka(1, :);        % Celý prvý riadok (ako menšia tabuľka)
Struct S.nazovPola
Používa sa bodková notácia.
meno_pacienta = pacient(1).meno; % Prístup k poľu 'meno' prvého pacienta

2.4 Logické Indexovanie (Filtrovanie)

Vytvorenie logického vektora (true/false) a jeho použitie na výber dát.

teploty = [15; 17; 12; 20];
index = teploty > 16;   % Vráti [false; true; false; true]
vysoke_teploty = teploty(index); % Vráti [17; 20]

% Priame použitie v jednom kroku pre tabuľky
nadpriemerne = pocasieTable(pocasieTable.Teplota > mean(pocasieTable.Teplota), :);

3. Operácie a Funkcie

3.1 Aritmetické Operácie

Maticové (Lineárna algebra): +, -, * (maticové násobenie), /, ^ (maticová mocnina), ' (transpozícia).

Po prvkoch (Element-wise): Používa sa bodka . pred operátorom. Vyžaduje matice rovnakých rozmerov.

A = [1 2; 3 4]; B = [2 0; 1 3];

% Maticové násobenie
C_mat = A * B; % Vráti [4 6; 10 12]

% Násobenie po prvkoch
C_elem = A .* B; % Vráti [2 0; 3 12]

% Umocnenie po prvkoch
A_moc = A.^2; % Vráti [1 4; 9 16]

3.2 Relačné a Logické Operácie

Používajú sa na porovnávanie a vytváranie logických indexov. Fungujú po prvkoch.

  • Relačné: > (väčší), < (menší), >= (väčší alebo rovný), <= (menší alebo rovný), == (rovná sa), ~= (nerovná sa).
  • Logické: & (logické A - AND), | (logické ALEBO - OR), ~ (logická negácia - NOT).

3.3 Funkcie pre Matice

A = [1 2 3; 4 5 6]; v = [10 20 30];
size(A)       % Vráti [2 3]
length(v)     % Vráti 3
sum(A)        % Vráti súčty stĺpcov: [5 7 9]
mean(v)       % Vráti 20
max(A)        % Vráti maximá stĺpcov: [4 5 6]

B = [1 2; 0 1];
inv(B)        % Vráti [1 -2; 0 1]

3.4 Funkcie pre Cell Array

mena = {'Jano', 'Katarina', 'Eva'};
dlzky = cellfun(@length, mena); % Vráti [4 8 3]

C = {123; 'text'};
celldisp(C) % Vypíše do konzoly: C{1} = 123, C{2} = 'text'

3.5 Funkcie pre Table

mena = {'Eva'; 'Jano'}; vek = [30; 25];
T = table(mena, vek);
sortrows(T, 'vek') % Zoradí tabuľku, Jano bude prvý

summary(T) % Vypíše do konzoly štatistiky pre stĺpec 'vek'

triedy={'1A';'1B';'1A'}; skore=[85;90;75];
studenti = table(triedy, skore);
groupsummary(studenti, 'triedy', 'mean', 'skore') % Vypočíta priemerné skóre pre každú triedu

3.6 Funkcie pre Struct

pacient.meno = 'Peter'; pacient.vek = 51;
fieldnames(pacient) % Vráti cell array {'meno'; 'vek'}

pacient_bez_veku = rmfield(pacient, 'vek');
% 'pacient_bez_veku' bude struct iba s poľom 'meno'

4. Manipulácia s Dátami

4.1 Pridávanie a Odstraňovanie

Pridanie stĺpca do tabuľky:

% 1. Pôvodná tabuľka
mena = {'Jano'; 'Eva'};
vek = [25; 30];
tabulka = table(mena, vek);
%
% tabulka = 
%     mena      vek
%     _______    ___
%     'Jano'     25 
%     'Eva'      30 

% 2. Pridanie stĺpca 'Vyska'
tabulka.Vyska = [180; 165];

% 3. Výsledná tabuľka
%
% tabulka = 
%     mena      vek    Vyska
%     _______    ___    _____
%     'Jano'     25     180  
%     'Eva'      30     165

Odstránenie dát (priradením prázdnej matice []):

% 1. Pôvodná tabuľka (z predošlého kroku)
%
% tabulka = 
%     mena      vek    Vyska
%     _______    ___    _____
%     'Jano'     25     180  
%     'Eva'      30     165 

% 2. Odstránenie stĺpca 'Vek'
tabulka.Vek = [];

% 3. Tabuľka po odstránení stĺpca
%
% tabulka = 
%     mena      Vyska
%     _______    _____
%     'Jano'     180  
%     'Eva'      165  

% 4. Odstránenie riadku, kde Vyska < 170
tabulka(tabulka.Vyska < 170, :) = [];

% 5. Finálna tabuľka
%
% tabulka = 
%     mena      Vyska
%     _______    _____
%     'Jano'     180

4.2 Spájanie Dát

Používajú sa hranaté zátvorky [], rovnako ako pri tvorbe.

A = [1; 2]; B = [3; 4];

% Horizontálne (vedľa seba)
H = [A, B]; % alebo [A B]   --> [1 3; 2 4]

% Vertikálne (pod seba)
V = [A; B]; %               --> [1; 2; 3; 4]

4.3 Konverzie medzi typmi

C = table2cell(T); % Table -> Cell Array
T = cell2table(C); % Cell Array -> Table
S = table2struct(T); % Table -> Struct
M = cell2mat(C);   % Cell Array (s číslami) -> Matica
