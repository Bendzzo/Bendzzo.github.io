1. OOP - Triedy a Dedičnosť

Každá trieda musí byť v samostatnom súbore NazovTriedy.m.

Rodič: Vozidlo.m

classdef Vozidlo
    properties
        znacka
        rychlost
    end
    methods
        % Konštruktor
        function obj = Vozidlo(znacka, rychlost)
            obj.znacka = znacka;
            obj.rychlost = rychlost;
        end
        
        function vypis(obj)
            fprintf('Vozidlo: %s\n', obj.znacka);
        end
    end
end

Potomok: Auto.m (Dedičnosť)

classdef Auto < Vozidlo
    properties
        spz
    end
    methods
        function obj = Auto(znacka, rychlost, spz)
            % 1. Volanie konštruktora predka (MUST HAVE)
            obj@Vozidlo(znacka, rychlost);
            % 2. Vlastné atribúty
            obj.spz = spz;
        end

        % Override (Prekrytie) metódy
        function vypis(obj)
            vypis@Vozidlo(obj); % Zavolá pôvodný výpis
            fprintf('ŠPZ: %s\n', obj.spz);
        end
    end
end

Potomok: Bicykel.m

classdef Bicykel < Vozidlo
    properties
        typ
    end
    methods
        function obj = Bicykel(znacka, rychlost, typ)
            obj@Vozidlo(znacka, rychlost);
            obj.typ = typ;
        end
    end
end

2. GUI (App Designer)

Všetko je v objekte app. Kód sa píše do Code View (Callbacks).

Načítanie hodnôt (Input)

% EditField (Numeric) -> vracia číslo
cena = app.CenaField.Value;

% EditField (Text) / DropDown -> vracia text
text = app.DPHDropDown.Value;

% Konverzia Text -> Číslo (KĽÚČOVÉ PRE VÝPOČTY)
percento = str2double(text);

Výpis hodnôt (Output)

% Label / Text Field -> musí dostať text
app.VysledokLabel.Text = 'Hotovo';

% Formátovanie čísla na text
vysledok = 125.50;
app.VysledokLabel.Text = sprintf('Cena: %.2f EUR', vysledok);

% Numeric Field -> musí dostať číslo
app.VysledokField.Value = vysledok;

Vzor Callback funkcie (Tlačidlo)

function ButtonPushed(app, event)
    % 1. Načítať
    vstup = app.VstupField.Value;
    
    % 2. Vypočítať
    vysledok = vstup * 2;
    
    % 3. Vypísať
    app.VystupLabel.Text = sprintf('%.2f', vysledok);
end

3. Algoritmy a Riadenie toku

For cyklus (pevný počet)

% Prechod cez vektor
vektor = [10 20 30];
for i = 1:length(vektor)
    prvok = vektor(i);
end

% Pevný rozsah
for k = 1:10
    disp(k);
end

While cyklus (podmienka)

x = 0;
while x < 5
    x = x + 1;
    % Beží kým podmienka platí
end

Podmienky (If-Else)

if x > 0
    disp('Kladné');
elseif x == 0
    disp('Nula');
else
    disp('Záporné');
end
