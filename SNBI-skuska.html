5. OOP - Triedy a Dedičnosť

Pozor: Každá trieda musí byť v samostatnom súbore s názvom NazovTriedy.m.

Rodič: Vozidlo.m

classdef Vozidlo
    properties
        znacka
        rychlost
    end
    methods
        function obj = Vozidlo(znacka, rychlost)
            obj.znacka = znacka;
            obj.rychlost = rychlost;
        end
        function vypis(obj)
            fprintf('Vozidlo: %s\n', obj.znacka);
        end
    end
end

Potomok: Auto.m

classdef Auto < Vozidlo
    properties
        spz
    end
    methods
        function obj = Auto(znacka, rychlost, spz)
            % 1. Volanie konštruktora predka
            obj@Vozidlo(znacka, rychlost);
            % 2. Vlastné atribúty
            obj.spz = spz;
        end
        % Override metódy
        function vypis(obj)
            vypis@Vozidlo(obj); % Volanie rodičovskej metódy
            fprintf('ŠPZ: %s\n', obj.spz);
        end
    end
end

Potomok: Bicykel.m

classdef Bicykel < Vozidlo
    properties
        typ
    end
    methods
        function obj = Bicykel(znacka, rychlost, typ)
            obj@Vozidlo(znacka, rychlost);
            obj.typ = typ;
        end
    end
end

6. GUI (App Designer)

Prístup ku komponentom je vždy cez app.NazovKomponentu.

Získanie hodnôt (Input)

% EditField (Numeric) -> vracia double
hodnota = app.CenaField.Value;

% EditField (Text) / DropDown -> vracia string/char
text = app.DPHDropDown.Value;

% Konverzia Text -> Číslo (DÔLEŽITÉ!)
cislo = str2double(text);

Výpis hodnôt (Output)

% Label / Text Field -> musí dostať text
app.VysledokLabel.Text = 'Hotovo';

% Formátovanie čísla na text
vysledok = 125.50;
app.VysledokLabel.Text = sprintf('Cena: %.2f EUR', vysledok);

% Numeric Field -> musí dostať číslo
app.VysledokField.Value = vysledok;

Callback funkcia (Tlačidlo)

function ButtonPushed(app, event)
    % 1. Načítať
    vstup = app.VstupField.Value;
    % 2. Vypočítať
    vysledok = vstup * 2;
    % 3. Vypísať
    app.VystupLabel.Text = sprintf('%.2f', vysledok);
end
